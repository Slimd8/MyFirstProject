# Lab Assignment # 27
This program reads the students name and scores, then calculates their average and give an letter grade.
## How to Run
Open new project in your IDE.
Name project "Student Grades"
Enter student names and scores.
View the letter grade output.
## Example Output
Student: Mike  
Average: 83.75 
Grade: B
If the file is missing: File not found
